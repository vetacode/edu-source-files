Prompt Auto Feed Artikel

Kumpulan prompt siap pakai untuk membuat sistem penarik artikel otomatis menggunakan HTML dan JavaScript.

Konsep

Apa Fungsi Prompt Ini?

Prompt ini digunakan untuk meminta AI membuat script HTML dan JavaScript yang mampu menarik artikel dari halaman lain dalam domain yang sama.

Teknik yang digunakan adalah fetch, DOMParser, dan querySelector.

Prompt 1

Prompt Dasar

Buatkan script HTML + JavaScript untuk menampilkan artikel dari halaman lain dalam domain yang sama menggunakan fetch dan DOMParser.

Sumber artikel:
https://suksespintar.biz.id/2/articles

Struktur HTML sumber:
- section article
- img
- h4 a sebagai judul dan link
- p sebagai deskripsi
- small sebagai tanggal

Output:
- tampilkan maksimal 6 artikel
- gunakan card layout sederhana
- responsive
- tampilkan gambar, judul, deskripsi, dan tanggal
- jika gagal tampilkan "Gagal memuat data"

Buatkan full HTML siap pakai.
Prompt 2

Prompt Profesional

Saya ingin membuat fitur auto feed artikel menggunakan HTML dan JavaScript tanpa backend.

Buatkan fullpage HTML dengan ketentuan:

1. Mengambil data dari halaman:
https://suksespintar.biz.id/2/articles

2. Gunakan:
- fetch()
- DOMParser
- querySelector

3. Ambil data:
- gambar
- judul
- link
- deskripsi maksimal 120 karakter
- tanggal

4. Tampilkan dalam bentuk:
- card grid responsive
- maksimal 6 artikel

5. Tambahkan:
- loading state
- error handling
- desain sederhana modern

6. Kode harus rapi dan mudah diedit.

Output: full HTML siap pakai.

Formula Prompt

Rumus Membuat Prompt Sendiri

Fitur tambahannya apa?

Kesimpulan

Inti Penggunaan Prompt

Kunci prompt yang bagus adalah menjelaskan sumber, struktur, data yang ingin diambil, dan tampilan akhirnya. Semakin jelas prompt, semakin siap pakai hasil HTML yang dibuat AI.
